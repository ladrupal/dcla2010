Windows Install-fest Error - Fix

Submitted by stevethewebguy on Sat, 2010-08-07 09:50.

So we got stuck after:

Installing WAMP.
Installing PHP 5.2.9-2
Creating a datbase through phpmyadmin.

THE FIX:

Create a user for that DB with one character name.
Example: s

Then go ahead and complete the Drupal install from http://localhost with the one character username and password you setup for it.

That will finish up getting Drupal installed on your laptop:)
