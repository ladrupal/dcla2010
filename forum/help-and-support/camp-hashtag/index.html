Something shorter??

Submitted by ChrisAldrich on Fri, 2010-07-30 14:52.

If we’re going to use a hashtag, partly by definition it should be:
SHORT, particularly for the networks (aka Twitter) where length is at a premium.
STANDARDIZED, something that everyone is using so the conversation can be followed (particularly if someone decides to create a track module on something like drupalcampla.com or the newly minted ladrupal.org it will be easier to suck in data from Twitter, Flickr, et al.
UNIQUE, something that isn’t being used in the same timeframe such that searched terms don’t provide overlapping and disparate conversations.

So why not something simple like #DxLA or #DCLA. Otherwise, I’m partial to #DrupalCamp to stay in line with convention while simultaneously (and though longer) #DrupalCampLA works well for the specific event while simultaneously showing up in searches for #DrupalCamp.

Chris Charlton, why don’t you anoint ONE as the official now, so it can groundswell into the camp?
