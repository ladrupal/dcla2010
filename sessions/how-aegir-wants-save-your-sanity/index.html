Aegir is this fantastic new way of deploying multi-site Drupal instances by using a Drupal instance to manage your Drupal Instances.

I plan to go over how to use Aegir, how to set up Aegir, and how to integrate it with your current workflow. I’ll probably go into how to expand on the base functionality that Aegir gives you – like all Drupal modules, there’s a lot to build on!

(I also plan to come up with a longer description for this, but give me a break, I’m on site today. :)

Note that there’s also a BOF proposed here: https://2010.drupalcampla.com/sessions/aeger-bof by https://2010.drupalcampla.com/users/kepford
