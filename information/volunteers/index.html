DrupalCamp LA 2010

Volunteers

Note: this is an archived site. Visit us at http://ladrupal.org.

DrupalCamp LA is a free event and entirely volunteer run (and no, this is not our day job); we expect active participation from everyone when it comes to running the conference. This is as simple as picking up after yourself or actively fixing something if you see it’s broken, but please feel welcome to take an active role in making this a great event.

We need your help during the camp. Please considering signing up for one of these volunteer sessions. Go to the Registration Room’s “Volunteer Table” for signing up, and instructions. More info will be posted, and made available at the Volunteer Table.

We have special needs for Set Up on Early Saturday morning, and Clean Up Late Sunday night (starting after the last session). If you come early and want to help, then let us know. We will keep you busy.

If you have Audio/Visual (AV) skills, each room needs the Speaker’s microphone set up and projector ready each morning, and first time speakers may need your assistance. Recording is done with Camtasia Software for audio and screen capture. We want Camtasia installed at least 30 minutes before the speaker’s laptop is plugged into the VGA cable.

Here is a brief list of some of the Volunteer Tasks available:

Set Up For Early Arrivers

Registration Room – Table and Chair Set Up
Registration Room – Name Badge Set Up
Registration Room – Goodie Bag Stuffing
Registration Room – Sign Set Up
Registration Room – Message Board Set Up
Registration Room – Jobs Wanted/Offered Board Set Up
Registration Room – Freebie Table Set Up

Sandwich Board Set Up – Move Boards from storage to the halls
Sandwich Board Signs – Move Signs onto boards as needed

Table Set Up – Set up tables on plaza area.

Audio/Video Operation – Presentation Room A/V Operator
Audio-Video Set Up – Set up in MDEA lecture hall
Double Check Halls – Check A/C and A/V in all lecture halls.
Troubleshooting – Technical Assistance in hardware or software

Balloon Filling – Fill balloons from tank, tie on string
Balloon Placement – Tie to Path to Parking and Lunch

Durring

Registration Table – Handles sign in sheet and name tags.
Room Clean Up – Walk rows picking up lost items and trash.
Room Trash Can Empty – Lecture Hall trash to dumpster
Raffle – Sell or hand out raffle tickets, run drawing, display and hand out prizes

Camtasia Recording Software Installation on Speakers’ Laptops
Troubleshooting – Technical Assistance in hardware or software

Stay late and Clean Up

Registration Room Tear Down – Pack boxes & signs, re-arrange tables/chairs
Registration Room Clean Up – Vacuum, trash bags to garbage cans

Sandwich Board Signs – Stack signs for next year
